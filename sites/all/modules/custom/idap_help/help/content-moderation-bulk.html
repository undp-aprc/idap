1. Navigate to http://www.inclusivedevelopmentasiapacific.net/admin/content/workflow.
  2. Tick items on the list that need to be moderated.
  3. From 'Operations' either:
    • Choose 'Delete Item' to completely remove an item.
    • Choose 'Change workflow state of post to new state' to change the contents' state.
    • Click 'execute' to carry out your selected action. If you chose 'Change workflow state of post to new state' you will be presented with some additional options.
  4. Change 'target state' to the new state you would like to set. If, for example, you would like to publish all the selected items, choose 'Final'.
  5. If you would like to change the default message, which is appended to the workflow log for each selected item' you can enter a new message in the 'Message' textbox.
  6. Click 'Next' to continue.
  7. Click 'Confirm' to complete the change.
