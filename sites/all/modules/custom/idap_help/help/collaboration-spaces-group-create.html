1. Navigate to http://www.inclusivedevelopmentasiapacific.net/node/add/group-collaboration.
  2. Name the Collaboration Space and provide a description. The name will be used as the Collaboration Space's title and the description will be displayed at the top of the Collaboration Space's page.
  3. Choose the content blocks that will be displayed on the Collaboration Space page.
  4. Choose the appropriate 'Group Options':
    • Group Visibility: Determines whether the Collaboration Space is visible to all (public) or only visible to space members (private).
    • Space Content Visibility: Determines whether content in the space is visible to all (public) or only visible to space members (private) by default.
    • Group roles and permissions: Always select the 'use default roles and permissions' option.
    • Theme: Choose the development theme this group belongs to.
    • Country: If this group relates to a particular country, select the country.
  5. Click 'Publish' to create the Collaboration Space.
