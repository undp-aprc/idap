Editing existing blocks

To edit an existing block to change the number of items listed or the type of content that is displayed, for example:

  1. Navigate to http://www.inclusivedevelopmentasiapacific.net/admin/content/blocks (this page is accessible from the 'Manage Blocks' link in administration toolbar at the top of the page).
  2. Scroll down the list of blocks to find the block you want to edit and click the 'edit' link to the right.
  3. Change the settings in the form, such as the title of the block, the number of items to display or the content type.
  4. Untick 'create new revision' then click 'Save'.
