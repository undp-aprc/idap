1. Navigate to the content administration page at http://www.inclusivedevelopmentasiapacific.net/admin/content
  2. You may use the controls in the 'show only content where' box to filter the list of content displayed by type and/or status. You can, for example, choose to display only content of type 'E-Discussion', only content that is 'published' or only content that is of type 'E-Discussion' and also published.
  3. In the content listing, each item has a checkbox to the left of the title. You can tick one or more checkboxes to select content upon which you wish to perform an operation.
  4. In the 'Update Options' box, you can choose an operation from the select list. When you click 'update' it will perform whichever operation you choose on each of the items of content you have checked below. If for example, you have checked three items, then choose 'Delete Selected Content' from the operations list, then click 'Update', it will delete the checked items (it will prompt you to confirm first!).
